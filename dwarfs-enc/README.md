# dwarfs-enc

A library for building and writing [DwarFS][dwarfs] archives (aka. DwarFS images),
building on top of [`dwarfs` crate][dwarfs-rs].

[dwarfs]: https://github.com/mhx/dwarfs
[dwarfs-rs]: https://crates.io/crates/dwarfs
